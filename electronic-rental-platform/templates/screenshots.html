Quick Installation Guide

Step 1: Extract Project

Extract the downloaded ZIP file and navigate to the project directory

cd rental_platform

Step 2: Create Virtual Environment

Create and activate a Python virtual environment

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

Step 3: Install Dependencies

Install all required Python packages

pip install -r requirements.txt

Step 4: Setup Database

Run migrations to create database tables

python manage.py makemigrations
python manage.py migrate

Step 5: Create Admin User

Create a superuser account for admin access

python manage.py createsuperuser

Step 6: Create Media Directories

Create directories for uploaded files

# Windows
mkdir media\products media\payments media\qrcodes

# macOS/Linux
mkdir -p media/products media/payments media/qrcodes

Step 7: Run Development Server

Start the Django development server

python manage.py runserver

Step 8: Access the Platform

Open your browser and navigate to:

Main Site: http://127.0.0.1:8000/
Admin Panel: http://127.0.0.1:8000/admin/
